HEX系列香水 主視覺

UI Design 20182019 iT 邦幫忙鐵人賽

以《Hex》為其名，一個魅惑人心的香水作為其系列品牌香水的主題。
故視覺上以六角形作為主元素，清新的主色系，是為了讓人可以輕易進入其中，透過魅惑的香氣使人不自覺得進入香氣的魔法之中，而不自知。

Plant Diary 主視覺

UI Design 20182019 iT 邦幫忙鐵人賽

《Plant Diary》 在種植植物的過程中，從種下種子那一刻起開始記錄他的點點滴滴，結合植物百科除了可以隨時查到種植的方法、了解我們所種植的植物，如果詳細記錄每天的水量、日照時數、溫度等更可以同步手機中的小植物讓他們一同成長，另外增加照相紀錄功能讓種植紀錄可以更完整有趣。

Do Not Disturb 主視覺

UI Design 20182019 iT 邦幫忙鐵人賽

《Do Not Disturb》以簡單的icon呈現，來表達自己多種願望想一次滿足的慾望！視覺用色上，則以想要安靜一個人為概念，所以色調上以沈穩舒眠的感覺來呈現。

Personal Skillsg 主視覺

UI Design 20182019 iT 邦幫忙鐵人賽
